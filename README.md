# 📖 Giới thiệu

Repository này ghi lại quá trình học lập trình **STM32** theo định hướng **Embedded Software Engineer**, từ những kiến thức nền tảng đến các ngoại vi thường dùng.

Mục tiêu của project không chỉ là viết được chương trình chạy trên STM32 mà còn hiểu:

* Cách phần cứng hoạt động
* Cách HAL Library được xây dựng
* Tư duy lập trình nhúng
* Debug và đọc Datasheet
* Thiết kế chương trình theo State Machine

Toàn bộ quá trình học được thực hiện trên **STM32CubeMX**, **STM32CubeIDE** và kit **STM32F205RBT6 ARM KIT**.
